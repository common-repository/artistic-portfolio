== Description ==
Create you own portfolio with multiple images! This plugin allows you to create portfolio items which will be displayed at specific page with Portfolio type. Portfolio page will be created automatically after creation of first portfolio item. Also each portfolio item supports tags and categories.

== Installation ==
1. Go to Plugins page
2. Click on Add New button 
3. Search for 'Artistic Portfolio'
4. Click on Install Now button for Artistic Portfolio plugin in search results
5. After plugin is installed click on Activate link for it
6. After plugin activation you will see Portfolio item on left menu in Admin mode.
